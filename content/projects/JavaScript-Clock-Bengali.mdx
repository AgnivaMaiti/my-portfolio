This project is a simple digital clock built with HTML, CSS, and JavaScript that displays the current time using Bengali numerals and formats the session as "পূর্বাহ্ণ" (AM) or "অপরাহ্ণ" (PM).

Check out the live demo: [Bengali Digital Clock](https://agnivamaiti.github.io/JavaScript-Clock-Bengali/)  
Explore the source code on GitHub: [JavaScript Clock Bengali](https://github.com/AgnivaMaiti/JavaScript-Clock-Bengali).
