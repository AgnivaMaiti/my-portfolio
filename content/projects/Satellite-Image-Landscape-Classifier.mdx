This project is a Tkinter-based Python application for classifying satellite images into different landscape categories using a VGG16-based Convolutional Neural Network (CNN). It features real-time annotation and a user-friendly interface to enhance usability.

Check out the project on GitHub: [Satellite Image Landscape Classifier](https://github.com/AgnivaMaiti/Satellite-Image-Landscape-Classifier)  
Watch the demo on YouTube: [Video Link](https://www.youtube.com/watch?v=kr7nHGzBT4c).
